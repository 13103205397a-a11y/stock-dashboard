# 小红书发布工作流

每天股市看板 17:00-18:00 更新后，运行：

```bash
node xhs/generate_xhs.js
python3 xhs/render_cards.py
```

脚本会读取：

- `logic.js`：产业链主线、瓶颈、风险
- `materials.js`：涨价方向、供需、下游、风险

输出：

- `xhs/output/YYYY-MM-DD-xhs.md`：可直接发布的小红书文案
- `xhs/output/YYYY-MM-DD-xhs.json`：结构化发布素材
- `xhs/output/YYYY-MM-DD-cards.html`：卡片预览页面
- `xhs/output/YYYY-MM-DD-cards/*.png`：可上传小红书的 6 张图

发布节奏：

- 主笔记：18:30-19:30
- 加更：20:30-21:30，仅当天有强题材时发

默认内容定位：

> A股题材看板：用更像小红书的观点卡片，拆今天资金在看什么、明天验证什么、哪里容易踩坑。

新版图片风格：

- 米白纸感底色
- 黑红强调
- 每张卡有“判断”栏
- 少堆数据，多给观点
- 封面必须有冲突感，例如“明天别急追”“别只看涨幅”

合规口径：

- 只写市场复盘、题材观察、风险提示
- 不写买入、目标价、必涨、梭哈、稳赚
- 不把个股写成明确推荐
